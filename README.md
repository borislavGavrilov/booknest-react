# booknest-react

This project is a React Single Page Application developed as a final exam assignment. It provides functionality for browsing and managing books, with public and private sections, user authentication, a catalog, detailed book pages, and full CRUD operations. Built using React, React Router, Context API, and Tailwind CSS.

How to To Run the Project :

1.Install dependencie - npm install

2.Start the React (client) app - npm run dev

3.Start the backend server - node ./server.js

4.Make sure both servers are running at the same time
